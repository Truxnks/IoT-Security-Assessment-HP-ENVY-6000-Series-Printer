# HP ENVY 6000 IoT Security Assessment

Professional PTES-based security assessment conducted against an HP ENVY 6000 Series All-in-One printer in a controlled home lab environment.

## Key Findings
- Critical IPP authentication bypass
- Unauthenticated XML information disclosure
- Weak management endpoint exposure

## Skills Demonstrated
- Network Enumeration
- Vulnerability Assessment
- HTTP/IPP Testing
- CVSS Scoring
- Risk Analysis
- Technical Reporting
- Remediation Planning

## Tools Used
- Nmap
- curl
- PowerShell
- Python
- searchsploit
